Unnamed Dash Game

This is a prototype game I made in unity.

The idea behind this game was a fun 2D platformer. The type that's really hard, but rewarding, where you beat a series of small levels, you die instantly and need to restart the whole level again.

I wanted the player to have a sense of action, but at the same time have patience and beat the levels through speed and precision.
So I put some thought into the designs of the prototype levels. They need to feel good while pertaining a sense of complexity. Of course this is just a prototype, so these are all sample levels. 3 Easy, followed by 3 Intermediate, followed by 3 Hard levels.

There are also 3 worlds planned, each world introducing a new mechanic, making the game become more and more intense as you need to combine these.

It is currently not available anywhere, but I have let many people play it as it's a project I'm quite proud of! (And I enjoy watching them struggle)
I am planning to continue working on it later, as I put the project on hold to wait for some new features to arrive in Unity. As the game is extremely dependant on input, the current InputManager is inadequate.
